# 안녕하세요!

포트폴리오 블로그의 첫 번째 글입니다.

## MDX란?

MDX는 마크다운 안에서 JSX(React 컴포넌트)를 사용할 수 있게 해주는 포맷입니다.

```tsx
function Hello() {
  return <h1>Hello, World!</h1>;
}
```

## 블로그 글 작성법

`src/content/blog/` 폴더에 `.mdx` 파일을 추가하면 자동으로 블로그에 표시됩니다.

frontmatter에 `title`, `description`, `date`, `tags`를 작성해주세요.
